# Templates for TS and Jest

## Dependencies
  - [TypeScript](https://www.typescriptlang.org/docs/)
  - [Jest](https://jestjs.io/)
  - [eslint](https://eslint.org/)
  - [prettier](https://prettier.io/)

## Usage

### Installation

```shell
asdf install
yarn install
```

### Scripts
  - `yarn start`
  - `yarn test`
  - `yarn lint`
